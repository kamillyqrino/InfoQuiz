# InfoQuiz 

O InfoQuiz é uma plataforma web interativa desenvolvida com o objetivo de auxiliar no aprendizado de informática por meio de quizzes dinâmicos e gamificados.

## Sobre o projeto
O sistema permite que usuários testem seus conhecimentos através de quizzes organizados por categorias e níveis de dificuldade, promovendo a fixação de conteúdos e a autoavaliação.

## Funcionalidades
- Cadastro e login de usuários  
- Recuperação de senha  
- Realização de quizzes interativos  
- Correção automática com pontuação  
- Feedback imediato após cada resposta  
- Acompanhamento de desempenho do usuário  
- Ranking de usuários  
- Painel administrativo para gerenciamento de quizzes  

## Tecnologias utilizadas
- PHP  
- HTML  
- CSS  
- JavaScript  
- MySQL  

## Conceitos aplicados
- Desenvolvimento web  
- Modelagem de sistemas (UML)  
- Banco de dados relacional  
- Requisitos funcionais e não funcionais  
- Interface responsiva  

## Público-alvo
Estudantes e profissionais da área de tecnologia que desejam revisar conteúdos e se preparar para provas e certificações.

## Objetivo
Oferecer uma solução prática, acessível e interativa para o ensino de informática, utilizando elementos de gamificação para aumentar o engajamento dos usuários.

 Como executar o projeto
1. Instalar o XAMPP ou outro servidor local  
2. Importar o banco de dados MySQL  
3. Colocar os arquivos na pasta do servidor (htdocs)  
4. Iniciar o Apache e MySQL  
5. Acessar pelo navegador  

## Diferenciais
- Sistema completo com autenticação de usuários  
- Feedback em tempo real  
- Relatórios de desempenho  
- Estrutura baseada em boas práticas de desenvolvimento
